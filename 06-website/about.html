关于本项目 / About This Project

本项目是一项关注职业测评信效度与整合应用的研究，重点考察 Gallup CliftonStrengths 与 Holland RIASEC 两种工具 在学生学业与职业发展中的可能作用。研究强调学术严谨性、批判性反思与证据等级区分。

研究目标 / Research Goals

  1. 系统梳理 Gallup CliftonStrengths 与 Holland RIASEC 的研究文献与信效度证据；
  2. 收集并整理测评结果样例与实践体系；
  3. 评估 Holland 职业集群分类的合理性与当代适用性；
  4. 设计并实施 TAG 标注系统，桥接 Gallup 优势主题与 Holland 兴趣类型；
  5. 设计多种整合思路，并通过 TAG 数据验证其理论合理性；
  6. 构建面向学生学业与职业发展的双维度综合评价体系；
  7. 开发研究成果展示网站。

证据等级说明 / Evidence Level Key

本项目对引用证据实行 A–D 四级分级：

  • A 强证据：多项独立大样本研究，方法严谨，跨文化复制。
  • B 中等证据：中等样本研究，方法基本合理，但未充分复制。
  • C 弱证据：小样本或方法学薄弱研究，理论推断。
  • D 非证据：机构宣传、专家意见、未经验证案例。

重要提醒 / Important Reminders

  • CliftonStrengths 与 Holland RIASEC 均非诊断工具，不可用于临床诊断或人事决策。
  • “信度高”不等于“效度高”，也不等于“有用”，三者需分别评估。
  • “类型匹配”不等于“职业成功”；职业选择受能力、价值观、机会结构、社会经济背景等多重因素影响。
  • 本项目产出的 TAG 映射与整合思路属于理论探索工具，除非有独立实证验证，否则不宣称其预测效度。
